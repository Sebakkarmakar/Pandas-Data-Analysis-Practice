# Liver Patient Data Analysis using Pandas

## Project Overview
This project focuses on performing complete data analysis using the Pandas library in Python on the Indian Liver Patient Dataset.

The project includes:
- Data loading
- Data cleaning
- Data exploration
- Filtering
- GroupBy operations
- Statistical analysis
- Handling missing values
- Visualization
- Merge/Join operations
- Pivot tables
- Advanced Pandas operations
